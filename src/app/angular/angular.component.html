Instalación de Angular

Para instalar Angular:

Asegúrate de tener Node.js instalado en tu sistema. Luego, ejecuta el siguiente comando en tu terminal:

npm install -g @angular/cli

Cómo crear componentes en Angular:

En tu terminal escribe el siguiente comando:

ng generate component nombre-del-componente

Comandos básicos en Angular:

Algunos comandos básicos que se utilizan en Angular incluyen:

  • ng serve: Inicia el servidor de desarrollo.
  • ng build: Compila la aplicación para producción.
  • ng test: Ejecuta pruebas unitarias.
  • ng generate: Genera componentes, servicios, módulos, etc.
  • ng lint: Verifica el código con linting.
Explicación de qué es Angular:

Es un framework de JavaScript, creado y mantenido por Google, es de código abierto, y está escrito en TypeScript. Su objetivo es desarrollar páginas SPA (single-page application) web de una sola página.
