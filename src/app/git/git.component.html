Comandos de Git


  1. git init:
    este comando creará un nuevo repositorio de git
    1
  2. git add:
    incluiremos los cambios al siguiente commit
    2
  3. git commit:
    establece un punto de control en el proceso de desarrollo al cual puedes volver más tarde si es necesario
    3
  4. git branch:
    renombra la rama actual
    4
  5. git push:
    mandaremos nuestro primer commit al repositorio remoto que hemos configurado
    5
